  • ObservableProperty

    A basic implementation of a property whose value can be observed using callbacks.

    Example:

var property = ObservableProperty(1)

property.subscribe { newValue in
    print("newValue: \(newValue)")
}

property.value = 2

// Executing this prints:
// "newValue: 2"
